Как играть в Road Shooter

Концепция гениально проста: один боец бежит по дороге. По пути вербуете новобранцев, проходите множительные ворота и сражаетесь с волнами врагов. Ваш отряд растёт от 1 до 200 бойцов.

3D-движок оживляет отряд плавными анимациями, динамичными ракурсами камеры и эффектами при падении врагов. Каждый забег ощущается как кино.

Краткое руководство

  • Свайпайте влево/вправо для управления отрядом.
  • Собирайте пикапы бойцов для пополнения.
  • Выбирайте ворота с умом: x2 (умножение), +N (добавление), рискованные (азарт).
  • Отряд стреляет автоматически — фокус на позиционировании и воротах.
  • Победите босса в конце каждого этапа.

Стратегия ворот: ключ к строительству отряда

Ворота — самый важный стратегический элемент. Каждый выбор определяет судьбу вашей армии.

Стратегии битв с боссами

3 эпических босса с уникальными паттернами атак. Менее 50 бойцов у босса — почти верная гибель.

Советы по выживанию

  • Размер отряда важнее всего. Бой с боссом — гонка DPS.
  • Рассредоточьте отряд. AoE-атаки заденут меньше бойцов.
  • Танки — ваш щит. Они поглощают смертельные удары.
  • Изучайте паттерны атак. Каждый босс телеграфирует удары.
  • Берегите улучшения для боссов. Максимальный эффект перед боем.

Система улучшений

Перманентные улучшения между забегами. Ресурсы ограничены — правильный порядок ускоряет прогресс.

Рекомендуемый приоритет

  1. Вместимость отряда — до 200, самое влиятельное.
  2. Урон — умножается на весь отряд.
  3. Начальный размер — раннее использование x2.
  4. Множитель бонуса ворот — накапливающийся эффект.
  5. Здоровье/Защита — снижает потери.

Бесконечный режим: покоряйте таблицу лидеров

После основных этапов открывается Бесконечный режим. Враги бесконечно усиливаются, каждое решение весит больше.

Стратегия

  • Не рискуйте после волны 50. Восстановление становится сложнее.
  • Приоритет x2 воротам. Только экспоненциальный рост спасает.
  • Разнообразие отряда. Подрывники для толп, снайперы для элит.
  • Замедление — для критических моментов.
